# 🤖 Agente Inteligente Híbrido: Gestión de Propiedad Intelectual

Este sistema es un prototipo avanzado que integra **IA Generativa**, arquitectura **RAG (Retrieval-Augmented Generation)** y flujos **ETL** para centralizar y consultar documentación técnica de propiedad intelectual de manera eficiente.

## 🌟 Logros Principales
- **Eficiencia en Búsqueda:** Reducción del **80%** en el tiempo de localización de documentos técnicos mediante búsqueda semántica.
- **Automatización Total:** Eliminación del **100%** del procesamiento manual de datos desde tablas HTML hacia bases de datos.
- **Optimización de Datos:** Procesamiento y estructuración de datasets técnicos para garantizar la calidad en modelos predictivos.

## 🛠️ Arquitectura del Sistema
El sistema se divide en tres capas principales diseñadas para la escalabilidad:

1.  **Capa de Ingesta (ETL):** Automatizada con **n8n**, encargada de extraer datos, procesar tablas HTML y cargarlos en PostgreSQL sin intervención humana.
2.  **Capa de Inteligencia:** Implementada en **Python**, utiliza modelos de lenguaje y bases vectoriales para la recuperación precisa de información técnica.
3.  **Capa de Interfaz:** Despliegue dual mediante una interfaz **Web** y un **Bot de Telegram** para consultas remotas en tiempo real.

## 💻 Stack Tecnológico
- **Lenguaje:** Python (Lógica RAG y procesamiento de lenguaje natural).
- **Orquestación:** n8n (Automatización de flujos de datos y ETL).
- **Bases de Datos:** PostgreSQL (Relacional) y MariaDB.
- **Visualización:** Dashboards en Power BI para el análisis de patrones y monitoreo.

## 🧩 Habilidades Aplicadas
- **Ingeniería de Prompts:** Optimización de respuestas del modelo de IA para contextos técnicos.
- **Arquitectura IoT:** Integración de nodos y sensores para la captura de datos en tiempo real.
- **Liderazgo Técnico:** Liderazgo en el diseño y ejecución de prototipos tecnológicos complejos.
